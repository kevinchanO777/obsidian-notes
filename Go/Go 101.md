#### Basic CLI

1. `go help`
2. `go version`
3. `go run`
4. `go build`
5. `go fmt`
6. `go vet`

#### 1. Hello world

*`hello-world.go`*
```go
package main

import "fmt"
  
func main() {

	fmt.Println("hello world")

}
```


```bash
# Run the program
go run hello-world.go

# Build the binary
go build hello-world.go

./hello-world
```
